> this file contains short cut key used in browser to search more efficiently

# short-Cut Keys
    -- cmd + t // opens a new tab in same window.
    -- ctrl + f // searches words in same pages
    -- cmd + shift + a // open search option for tabs i.e tab search
    -- ctrl + tab // jumps into next tab in same browser, + shift // jumps to previous tabs
    -- cmd + opt + arrows // manually navigate through tabs
    -- cmd + l // to jump directly to the url bar
    -- cmd + w // closes the current tab
    -- cmd + ctrl + f // full screens the application windows
